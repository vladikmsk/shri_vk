Задание №1

Перечислите известные вам инструменты и приёмы для отладки кода в разных браузерах.

Из увиденных дебагеров:
Internet Explorer - Developer Tools
Firefox - Firebug
Chrome - Developer Tools

Основные приемы:
1. Установка breakpoint в отладочном коде, затем пошаговое выполнение. Возможно применять команды в console для интерактивности;
2. Использование своей функции с аргументом event и командой debugger в теле функции;
3. "Умная" установка breakpoint'ов за счет применения __defineSetter__ и __defineGetter__;
4. Для просмотра значений переменных в нужном месте кода можно использовать стандартные console.log(_var), alert(_var);

Для просмотра печатной версии войдите в Файл->Предварительный просмотр или нажмите на Печатать
