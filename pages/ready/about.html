Опыт разработки:

Вопрос: Расскажите нам о вашем опыте разработки. Ничего не упускайте: это может быть как серьезный интерфейс, так и просто домашняя страничка:
Ответ: утилита для удаленного управления звуком ОС Windows XP (.NET 2.0), сайт объявлений для рекламного агентства (ASP.NET, T-SQL от А до Я) показать не получится, сайт прикрыли www.rb-vdv.ru, сайт конкурса красоты miss.mtcs.ru с голосовалкой, тонкий клиент для MS Navision (ASP.NET, T-SQL), утилита для ICMP запросов по списку с файла и формирование отчета о состоянии хостов (C#), страничка для генерации паролей (PHP).

Вопрос: Какие языки программирования, фреймворки и технологии вы использовали? Сложились ли у вас личные предпочтения?
Ответ: C#, C++, Pascal, PHP, Perl, T-SQL, PL/SQL, SQL, JavaScript. Из фреймворков: .NET 2.0. К разным языкам, платформам отношусь абсолютно нейтрально, больше интересует подход к решению проблемы.

Вопрос: Какие редакторы и среды разработки вы использовали? Чем обычно пользуетесь?
Ответ: Delphi, Visual Studio 2005. Обычно пользуюсь FARом.

Вопрос: С какими ОС приходилось сталкиваться? Для каких целей вы их использовали?
Ответ: Windows Server 2003,2008, Windows XP, Vista, 7. Linux CentOS, Mandriva, Ubuntu Desktop/Server. FreeBSD. VxWorks. Cisco IOS. Windows Server 2003/2008 для администрирования корпоративных сервисов, кластеров, биллинговых систем, а также телефония Cisco CCM. Windows XP,7,Vista для просмотра фильмов. Linux CentOS, Mandriva, Ubuntu Server для администрирования корпоративных сервисов(сайты, почтовые сервера), развертывания биллинговых систем Lanbilling, Bill-on-net, а также для развертывания SMS-центров и VoIP(Asterisk). FreeBSD для развертывания систем IDS/IPS. VxWorks для модулей земных станций спутниковой связи. Cisco IOS для сетей (маршрутизирование статическое/динамическое, firewall, VoIP через CME), иногда приходилось расширять функционал за счет TCL.

Вопрос: Приходилось ли вам пользоваться системами контроля версий? Если да, то какими? Какая из них вам нравится больше всего?
Ответ: Пользоваться не приходилось, но игрался с CVS, SVN. Удобно было пользоваться Tortoise клиентом для SVN.

Вопрос: Использовали ли вы системы постановки и ведения задач?
Ответ: Что то не припоминаю.

Вопрос: Какие полезные для фронтенд-разработчика ресурсы вы используете и можете посоветовать?
Ответ:sdfsdfsdf

Вопрос: Работали ли вы где-нибудь? Какие у вас были должностные обязанности? Есть ли опыт работы в команде?
Ответ: Опыта работы в команде нет. Разработки все велись в одиночку. А так работал в основном инженером. Обслуживал Земные станции спутниковой связи Gilat, SatNet, занимался сетями, VoIP и Видеоконференцсвязь (Polycom, Tandberg, VCON).
